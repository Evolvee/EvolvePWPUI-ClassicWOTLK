# ArenaMarker

Gives your group class-based Raid Markers in arena.
<br>
<br>
<br>
- [x] Class specific marks
- [x] Customizable options to mark pets when summoned
- [x] Prioritize which marker(s) you want for pets
- [x] Classic & Retail support
